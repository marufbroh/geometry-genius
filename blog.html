1. What is DOM and what is the purpose of DOM?

Answer: DOM stands for Document Object Model. CSS and HTML are viewed as objects in JavaScript. JavaScript has access to any property of an object, so it converts HTML and CSS into a Document Object Model, so it can modify the document structure, style, and content as needed.

2. How will you select HTML elements using DOM? Name the DOM methods?

Answer: DOM is capable of selecting any HTML element and changing its value. In Dom, we can select HTML elements using 5 methods.

  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

3. What is event bubble?

Answer: A target element runs handlers when an event occurs, then its parent, then its parent's parents, and so on until all actions are completed. Its going to up and up like a bubble..Thats why people called it event bubble. Its work for all of handlers events.

4. What is a callback function and why will you use it?

Answer: A callback function in javascript is a function that can pass another function as a parameter and calls it inside. It will be executed after the completion of the first function.
